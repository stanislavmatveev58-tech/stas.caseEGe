🧮 Алгебра

Формулы сокращённого умножения

(a+b)² = a² + 2ab + b²
(a−b)² = a² − 2ab + b²
a² − b² = (a−b)(a+b)
(a+b)³ = a³ + 3a²b + 3ab² + b³
(a−b)³ = a³ − 3a²b + 3ab² − b³
a³ + b³ = (a+b)(a² − ab + b²)
a³ − b³ = (a−b)(a² + ab + b²)

Арифметическая прогрессия

aₙ = a₁ + (n−1)d
Sₙ = (a₁ + aₙ)·n / 2 = (2a₁ + (n−1)d)·n / 2

Геометрическая прогрессия

bₙ = b₁·qⁿ⁻¹
Sₙ = b₁·(qⁿ − 1)/(q − 1), q ≠ 1
Сумма бесконечной убывающей: S = b₁/(1−q), |q|<1

Свойства степеней

aᵐ·aⁿ = aᵐ⁺ⁿ
aᵐ / aⁿ = aᵐ⁻ⁿ
(aᵐ)ⁿ = aᵐⁿ
(ab)ⁿ = aⁿ·bⁿ
(a/b)ⁿ = aⁿ / bⁿ
a⁰ = 1, a⁻ⁿ = 1/aⁿ

Свойства корней

√(ab) = √a·√b
√(a/b) = √a / √b
(√a)² = a
√(a²) = |a|
ⁿ√(aᵐ) = aᵐ⁄ⁿ

Свойства логарифмов

logₐ(xy) = logₐx + logₐy
logₐ(x/y) = logₐx − logₐy
logₐxᵏ = k·logₐx
logₐb = 1 / log_b a
logₐb = log_c b / log_c a
a^(logₐb) = b

Квадратные уравнения

ax² + bx + c = 0
D = b² − 4ac
x₁₂ = (−b ± √D) / (2a)
Теорема Виета: x₁ + x₂ = −b/a, x₁·x₂ = c/a

Разложение на множители

ax² + bx + c = a(x − x₁)(x − x₂)

📐 Планиметрия

Площадь треугольника

S = ½·a·hₐ
S = ½·ab·sin γ
S = √(p(p−a)(p−b)(p−c)) (Герон)
S = abc/(4R)
S = pr

Теорема синусов

a/sin A = b/sin B = c/sin C = 2R

Теорема косинусов

a² = b² + c² − 2bc·cos A

Прямоугольный треугольник

Теорема Пифагора: a² + b² = c²
Высота из прямого угла: h² = a₁·b₁
Радиус вписанной: r = (a+b−c)/2
Площадь: S = ab/2

Равносторонний треугольник

h = a√3/2
S = a²√3/4
R = a√3/3, r = a√3/6

Четырёхугольники

Прямоугольник: S = ab, P = 2(a+b)
Квадрат: S = a², d = a√2
Параллелограмм: S = a·hₐ = ab·sin α
Ромб: S = a·h = ½·d₁·d₂
Трапеция: S = (a+b)/2 · h, средняя линия = (a+b)/2

Окружность и круг

Длина окружности: C = 2πR
Площадь круга: S = πR²
Длина дуги: L = πR·α/180°
Площадь сектора: S = πR²·α/360°

Длина биссектрисы

l = (2ab·cos(γ/2))/(a+b) или l = √(ab − a₁b₁), где a₁,b₁ — отрезки стороны

Длина медианы

mₐ = ½√(2b² + 2c² − a²)

🧊 Стереометрия

Куб, призма, параллелепипед

V = Sосн·h
Прямоугольный параллелепипед: V = abc
Куб: V = a³, S = 6a²
Призма: Sполн = Sбок + 2Sосн

Цилиндр и конус

Цилиндр: V = πR²h, Sбок = 2πRh, Sполн = 2πR(R+h)
Конус: V = ⅓πR²h, Sбок = πRl, где l = √(R²+h²)

Пирамида и шар

Пирамида: V = ⅓Sосн·h
Шар: V = ⁴⁄₃πR³, S = 4πR²

📐 Тригонометрия

Основные тождества

sin²α + cos²α = 1
tgα = sinα/cosα, ctgα = cosα/sinα
1 + tg²α = 1/cos²α, 1 + ctg²α = 1/sin²α

Формулы сложения

sin(α±β) = sinα·cosβ ± cosα·sinβ
cos(α±β) = cosα·cosβ ∓ sinα·sinβ
tg(α±β) = (tgα ± tgβ)/(1 ∓ tgα·tgβ)

Формулы двойного угла

sin2α = 2sinα·cosα
cos2α = cos²α − sin²α = 2cos²α − 1 = 1 − 2sin²α
tg2α = 2tgα/(1−tg²α)

Преобразование суммы в произведение

sinα + sinβ = 2sin((α+β)/2)·cos((α−β)/2)
cosα + cosβ = 2cos((α+β)/2)·cos((α−β)/2)
sinα − sinβ = 2cos((α+β)/2)·sin((α−β)/2)
cosα − cosβ = −2sin((α+β)/2)·sin((α−β)/2)

Таблица значений

0°: sin=0, cos=1, tg=0
30°: sin=½, cos=√3/2, tg=√3/3
45°: sin=√2/2, cos=√2/2, tg=1
60°: sin=√3/2, cos=½, tg=√3
90°: sin=1, cos=0, tg не определён

🎲 Вероятность

Основные понятия

P(A) = m/n (классическая вероятность)
P(A) + P(¬A) = 1
P(A∪B) = P(A) + P(B) − P(A∩B)
Для независимых: P(A∩B) = P(A)·P(B)
Формула Бернулли: Pₙ(k) = Cₙᵏ·pᵏ·qⁿ⁻ᵏ

📈 Производные

Правила дифференцирования

(c)' = 0
(xⁿ)' = n·xⁿ⁻¹
(u ± v)' = u' ± v'
(uv)' = u'v + uv'
(u/v)' = (u'v − uv')/v²
(sin x)' = cos x
(cos x)' = −sin x
(tg x)' = 1/cos²x
(ctg x)' = −1/sin²x
(eˣ)' = eˣ
(aˣ)' = aˣ·ln a
(ln x)' = 1/x

Первообразные

∫xⁿ dx = xⁿ⁺¹/(n+1) + C, n ≠ -1
∫1/x dx = ln|x| + C
∫sin x dx = −cos x + C
∫cos x dx = sin x + C
∫eˣ dx = eˣ + C
∫aˣ dx = aˣ/ln a + C
